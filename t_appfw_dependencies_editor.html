Adding app source dependencies to use in the QRadar App Editor

You can use third-party dependencies, such as RPMs or Python libraries, when you are developing an application in the IBM® QRadar® App Editor.

About this task

Add third-party dependencies to your application by adding them to the src_deps subdirectory of the app folder. For more information, see Source dependencies.

Procedure

  1. Add the source dependencies to the src_deps subdirectory of the app folder on your local computer.
    Tip: The src_deps directory is an optional directory that developers can add to the app root folder to add dependencies to an app. For more information, see App file structure.
  2. Upload your application by using the procedure described in Editing apps in the IBM QRadar App Editor.
    Tip: While the editor does not support live editing of source dependencies, any existing dependencies are installed and available for use in the editor. The src_deps directory is not shown in the App Editor, but its contents are included in your application when you export or redeploy it.
